# LFSTray
This ia a simple standalone systray.


Forked from stalonetray ( big thanks )

Original Code here:

https://github.com/kolbusa/stalonetray



**To build/install:**
```console
./autogen.sh --prefix=/usr
make
make install
```


**QUICK USE:**  
lfstray -h

lfstray -t --tint-level 130 --tint-color "#ffffff"
